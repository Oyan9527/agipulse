The signal is buried in noise

The AI field produces far more each day than anyone can read — official blogs, papers, open-source releases, tech media and community chatter across hundreds of channels. AGI Pulse exists to filter signal from noise: it reads all of it, so you only see what's worth your time.

技术架构
How it works

零后端，全自动

Zero backend, fully automated

整站没有服务器、没有数据库。定时任务运行数据流水线，产出静态文件，由 GitHub Pages 直接托管。几乎零成本运行、无需运维、零用户数据。

No servers, no database. A scheduled job runs the pipeline, emits static files, and GitHub Pages serves them. Near-zero cost, no ops, no user data.

GitHub Actions（每 4 小时） └─ Python 数据流水线 └─ 生成静态 JSON └─ 提交回仓库 └─ GitHub Pages 托管，前端读取渲染
GitHub Actions (every 4 hours) └─ Python pipeline └─ static JSON └─ committed to repo └─ served by GitHub Pages

Strict only for general-tech sources; native AI sources pass through.
4
两级 LLM 处理（DeepSeek）
Two-stage LLM (DeepSeek)
低成本粗筛剔噪，再四维度打分：权威 30% + 新颖 25% + 影响 25% + 实用 20%，并生成分类、中文摘要、深度评分与译题。
Cheap prefilter, then 4-dim scoring (authority 30% + novelty 25% + impact 25% + practicality 20%), plus category, Chinese summary, depth score and translation.
5

Bag-of-words similarity + model/product identifiers collapse one event into one card.

≥2 sources OR score ≥0.72 to be curated; quiet days aren't padded.
